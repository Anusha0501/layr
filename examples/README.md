# Examples

This directory contains sample project plans generated by **Layr – AI Planning Layer**.

These examples demonstrate the kind of structured, actionable Markdown plans Layr produces
when given a natural language project description.

## Purpose

These examples help:
- New users quickly understand what Layr generates
- Contributors understand the extension’s core functionality
- Readers see real-world use cases before installing the extension

Each plan follows Layr’s typical output structure, including project overview,
features, technology stack, development steps, and enhancements.

You can use these examples as inspiration for your own projects or as references
when exploring Layr’s capabilities.
